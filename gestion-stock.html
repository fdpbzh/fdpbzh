Gestion du stock

Outil privé de l'asso — page non indexée, ne partage pas ce lien.
Comment ça marche : une vente (en ligne ou en espèces) = tu retires 1 à la bonne case avec le bouton − (ou tu tapes la nouvelle quantité). Quand tu as fini, clique « Télécharger stock.json », puis dans GitHub Desktop : glisse le fichier téléchargé dans le dossier assets (il remplace l'ancien) → Commit → Push. Le site se met à jour en 1–2 min. Pense à décompter les ventes Stripe ici aussi (elles ne s'enlèvent pas toutes seules).
