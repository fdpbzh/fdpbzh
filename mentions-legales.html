Mentions légales

Le site fdp.bzh est volontairement minimaliste : pas de tracker publicitaire, pas de cookie, pas de partage de données. Voici quand même les détails utiles.

Hébergement

Base de données scores : Supabase (Frankfurt, Allemagne — UE).

Données collectées

Le site collecte uniquement les données suivantes via le jeu La Moule Volante :

  • Pseudo (12 caractères max) — choisi par le joueur, public dans le classement
  • Score — distance atteinte, public dans le classement
  • Pseudo Instagram (optionnel) — uniquement si le joueur le saisit, utilisé pour identifier les gagnants. Non public.

Aucun cookie n'est déposé. Aucun tracker tiers n'est utilisé. Aucune donnée n'est partagée avec des partenaires.

Toutes les données seront supprimées après le 31 août 2026 (fin de la fête + une semaine).

Tes droits (RGPD)

Conformément au RGPD, tu peux à tout moment :

  • Demander l'accès à tes données
  • Demander leur rectification
  • Demander leur suppression

Pour exercer ces droits, écris-nous à lassoduport@gmail.com en précisant ton pseudo et la date approximative de ta partie.
